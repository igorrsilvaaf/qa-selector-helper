# Política de Privacidade - QA Selector Helper

**Última atualização:** 04/01/2025

## Coleta de Dados

A extensão **QA Selector Helper** preza pela sua privacidade.

- **Não coletamos** dados pessoais, senhas, histórico de navegação ou informações sensíveis.
- **Não enviamos** dados para servidores externos. Todo o processamento é feito localmente no seu navegador.

## Uso de Permissões

1. **activeTab / All URLs**: Necessário apenas para ler o DOM da página que você está inspecionando e gerar os seletores CSS/XPath. Nenhum dado da página é armazenado permanentemente.
2. **Storage**: Usado exclusivamente para salvar suas preferências locais (ex: tema Dark/Light, último seletor copiado) dentro do próprio navegador.
